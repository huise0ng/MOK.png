## UZU-우주
나 그리고 친구에게 미래에 전달되는 편지 웹뷰 앱이에요. 
메시지, 오프라인 편지가 있지만 우주는 앞과 다르게 시간을 설정하고, 미래에 편지를 읽을 수 있다는 차별점이 있어요.

## 기능
아래의 우주의 기능들이에요.
- 미래의 나를 위한 편지를 쓸 수 있어요.
- 친구에게 미래 편지를 공유할 수 있어요.
- 카카오톡 친구에게 미래 편지를 공유할 수 있어요.
